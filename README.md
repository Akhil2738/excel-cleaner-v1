# Excel Cleaner – V1

A simple automated Excel cleaning tool that can:

- Remove duplicates  
- Remove empty rows  
- Trim extra spaces  
- Standardize text  
- Apply basic cleanup rules  
- Export a clean Excel file  

## Tech Stack
- Python  
- Pandas  
- OpenPyXL

Version 1 – Basic cleaner created.
